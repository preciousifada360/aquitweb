                                                                                                                                    Malik Shailesh Tiwari 7 months ago

                                                                                                                                    Well, the unexpected "T_Variable" error some times occur in php even if your php code is written perfectly fine.
                                                                                                                                    The main reason behind it is the empty spaces in your php files that store the code, sometimes these empty spaces act as a character that throws error on compilation.
                                                                                                                                    To overcome this please remove all the empty spaces in your php files ("also try to avoid deleting the script, obviously :)") also with priority first remove the empty spaces before $update_query and you will see the change.
                                                                                                                                    Hope it helps !
